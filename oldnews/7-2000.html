July 9th, 2000 [Phrodide] Apollo v0.01c Released!

Apollo version 0.01c has been released! Here's what's new:
- Multi-lingual support
- Plugin support (very preliminary)
- Exception code rewritten (finally)
- RSP, RDP, and Audio dummy support added, next release will have more.
- Speed hack --some roms run too fast while others no change.
- Seriously cleaned up some graphics code.

Note to webmasters: Please do not upload the binary to your own server, but instead link to our download page, http://apollo.emulation64.com/download.htm. We want to help Emulation64 and this is the best way.


July 8th, 2000 [Phrodide] Small Update

Just wanted to let everyone know that we are not dead, and everything is going well. Version C, planned for on about the first week of July, has been steadily growing with more and more features that we didn't plan on including. So Version C may be delayed a little longer, or may be out today. It all mostly depends on where we decide to stop.

Our one little post about Apollo being translated into French has prompted others to join in. So far we have English, French, Spanish, Dutch, Danish, Swedish, and Portuguese. German is lined up to be included on release too.
