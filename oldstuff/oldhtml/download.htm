This project started April 1st, 2000, so only time will tell on how fast, how compatable, and how worthy this emulator will be.

Projected requirements for this emulator are as follows:

  • Pentium II machine or AMD K6-2, speed not determined yet.
  • 32Megs of RAM for most roms, 64MB for a few (RE, Donkey Kong)
  • 3D video not required, but strongly suggested. 3DFX, nVidia, and ATI cards welcome.

Do not expect miracles, please. If you want a nice emulator, try Nemu, UltraHLE or Corn.
